

# Mobile Swipe Navigation + PWA Support

Two features in one: swipe-based navigation for mobile users and full Progressive Web App support.

## 1. Swipe Navigation (Slide Left-to-Right)

Mobile users will be able to swipe left-to-right to go back to the previous page (like native iOS/Android back gesture). A visual edge indicator will show swipe progress.

**How it works:**
- A `SwipeBackNavigator` wrapper component detects horizontal touch gestures on mobile
- Swiping from the left edge (within 30px) toward the right triggers `history.back()`
- A subtle visual indicator (gradient edge overlay) shows swipe progress
- Only activates on mobile devices; ignored on desktop
- Uses `framer-motion` (already installed) for the visual feedback animation
- Won't interfere with scrollable content or horizontal carousels

**Files:**
- `src/components/mobile/SwipeBackNavigator.tsx` -- New component
- `src/components/layout/MainLayout.tsx` -- Wrap children with SwipeBackNavigator

## 2. PWA Support

Make the app installable from the browser with offline caching, app manifest, and service worker.

**What users get:**
- Install the app to their home screen (like a real app)
- Works on both iPhone and Android
- Offline support with cached assets
- App icon, splash screen, and standalone mode (no browser chrome)

**Technical approach:**
- Install `vite-plugin-pwa` package
- Configure in `vite.config.ts` with manifest, icons, and workbox settings
- Add PWA meta tags to `index.html` (apple-touch-icon, theme-color, etc.)
- Create PWA icon files in `/public` (192x192 and 512x512)
- Add `/~oauth` to `navigateFallbackDenylist` to prevent caching OAuth redirects
- Create an optional `/install` page with install prompt

**Files:**
- `vite.config.ts` -- Add VitePWA plugin configuration
- `index.html` -- Add PWA meta tags and apple-touch-icon links
- `public/manifest.json` -- Auto-generated by plugin, but icons needed
- `public/pwa-192x192.png` -- App icon (will use a generated placeholder)
- `public/pwa-512x512.png` -- App icon large
- `src/hooks/usePWAInstall.ts` -- New hook to handle install prompt
- `src/pages/InstallPage.tsx` -- New page with install instructions and prompt button
- `src/App.tsx` -- Add /install route
