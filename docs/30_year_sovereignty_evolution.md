## 5. Emergency Rollback Scenarios

### Scenario A: Governance Capture Detected
1. Capture protection triggers alert (>60% risk score)
2. All proposals frozen
3. Voting weights dynamically reduced for dominant entities
4. Emergency governance review convened
5. If unresolved: rollback to previous phase

### Scenario B: Systemic Crisis
1. Crisis simulation flags critical risk
2. Multi-region confirmation required (3+ regions)
3. Founder emergency control activated
4. Cross-border capital routing frozen
5. Sovereignty progression locked
6. Phase rollback to Level 0 if necessary

### Scenario C: Constitutional Violation Attempt
1. Constitutional layer blocks violating proposal
2. Immutable invariant violation logged
3. Proposing entity trust score penalized
4. Governance council emergency session triggered
5. Attempting entity sovereignty level frozen
